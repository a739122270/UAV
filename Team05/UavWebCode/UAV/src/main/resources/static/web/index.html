Welcome to UAV Fleet Management!



Comparing with single UAV, the UAV fleet has more advantages such as larger control areas, interactive cooperation and task allocation.
Therefore, clustering is a significant trend of the UAV development.



Same Account for Web client and Android client.

When using the web version to perform formation path planning tasks, please log in to the same account on the app and the web.
Use the app to link your UAV.


Create flight tasks using the web client only.

The system automatically generates a path through these coordinates in order. If it is a fleet task, the captain plans the route of the host.
The system will automatically generate flight routes for other wingmen.
Information such as flight time and route will be recorded in the database for easy viewing.


Create a team and natrually become the team manager.

The drone of the team manager defaults to the host.
When creating a team, you can add players by searching for other usernames.
Not all team members must participate in every mission.
